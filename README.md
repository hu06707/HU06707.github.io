## GitHub Pages部署指南

### 1. 在GitHub上创建仓库
1. 登录GitHub账号
2. 点击右上角的「+」按钮，选择「New repository」
3. 仓库名称设置为：`hwb-personal-website`（或其他你喜欢的名称）
4. 勾选「Initialize this repository with a README」
5. 点击「Create repository」

### 2. 克隆仓库到本地
```bash
git clone https://github.com/your-username/hwb-personal-website.git
cd hwb-personal-website
```

### 3. 添加项目文件
将以下文件复制到仓库目录：
- index.html
- style.css
- script.js

### 4. 提交并推送代码
```bash
git add .
git commit -m "Initial commit of personal website"
git push origin main
```

### 5. 启用GitHub Pages
1. 进入仓库的「Settings」页面
2. 向下滚动到「GitHub Pages」部分
3. 在「Source」下拉菜单中，选择「main」分支
4. 点击「Save」按钮
5. 等待几分钟后，你的网站将可以通过以下URL访问：
   `https://your-username.github.io/hwb-personal-website/`

## 本地运行

## 自定义修改

1. 打开`index.html`文件
2. 修改相关内容：

   - 联系方式

### 修改交互效果
1. 打开`script.js`文件
